# 형변환

형변환은 값의 자료형을 다른 자료형으로 바꾸는 것을 말합니다. 예를 들어, 문자열 "123"을 숫자 123으로 바꾸거나, 숫자 1을 참거짓 값인 `참`으로 바꿀 수 있습니다.

## 숫자로 바꾸기

문자열이나 참거짓 값을 숫자로 바꿀 때는 `숫자로 바꾸기`를 사용합니다.

<code-runner :code='`"123"을 숫자로 바꾸기 보여주기
참을 숫자로 바꾸기 보여주기
거짓을 숫자로 바꾸기 보여주기`' />

- 숫자로 바꿀 수 없는 문자열(예: "안녕")을 숫자로 바꾸려고 하면 오류가 발생합니다.
- `참`은 `1`로, `거짓`은 `0`으로 바뀝니다.

## 문자열로 바꾸기

어떤 값을 문자열로 바꿀 때는 `문자열로 바꾸기` 또는 `문자로 바꾸기`를 사용합니다.

<code-runner :code='`123을 문자열로 바꾸기 보여주기
참을 문자로 바꾸기 보여주기`' />

::: tip 보여주기와 형변환
`보여주기`는 값을 화면에 출력할 때 자동으로 문자열로 변환하여 보여줍니다. 따라서 단순히 출력만 하려는 경우에는 따로 형변환을 할 필요가 없습니다. :::

## 참거짓으로 바꾸기

값을 참거짓(불리언) 값으로 바꿀 때는 `참거짓으로 바꾸기` 또는 `불리언으로 바꾸기`를 사용합니다.

<code-runner :code='`1을 참거짓으로 바꾸기 보여주기
0을 불리언으로 바꾸기 보여주기
"참"을 참거짓으로 바꾸기 보여주기
""을 불리언으로 바꾸기 보여주기`' />

- **숫자:** `0`은 `거짓`이 되고, 그 외의 모든 숫자는 `참`이 됩니다.
- **문자열:** `"참"`, `"true"`, `"맞음"`은 `참`이 되고, `"거짓"`, `"false"`, `"아님"`은 `거짓`이 됩니다. 그 외의 비어 있지 않은 문자열은 모두 `참`이 되며, 빈 문자열(`""`)은 `거짓`이 됩니다.
