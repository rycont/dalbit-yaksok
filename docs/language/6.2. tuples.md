# 튜플

튜플은 여러 개의 값을 하나로 묶어서 다룰 수 있는 자료구조입니다. 목록과 비슷하지만, 한 번 만들어지면 그 내용을 바꿀 수 없다는 점이 다릅니다.

## 튜플 만들기

튜플은 소괄호 `()` 안에 값들을 쉼표로 구분하여 넣어서 만듭니다.

<code-runner code='좌표 = (10, 20)
좌표 보여주기' />

값이 하나만 있는 튜플을 만들 때는 값 뒤에 반드시 쉼표를 붙여야 합니다. 쉼표가 없으면 단순한 소괄호로 인식됩니다.

<code-runner code='단일_값 = (42,)
단일_값 보여주기' />

비어 있는 튜플은 다음과 같이 만듭니다.

<code-runner code='빈_튜플 = ()
빈_튜플 보여주기' />

## 값 가져오기

튜플의 각 값은 대괄호 `[]`와 인덱스를 사용하여 가져올 수 있습니다. 인덱스는 0부터 시작합니다.

<code-runner :code='`색상 = ("빨강", "초록", "파랑")
색상[0] 보여주기
색상[1] 보여주기
색상[2] 보여주기`' />

## 함수와 함께 사용하기

튜플은 함수의 인자로 전달하거나, 함수에서 여러 개의 값을 한꺼번에 반환할 때 유용하게 사용됩니다.

<code-runner :code='`약속 (x, y) 이동하기
    (x + 10, y + 10) 반환하기

현재_위치 = (0, 0)
다음_위치 = 이동하기(현재_위치[0], 현재_위치[1])
다음_위치 보여주기`' />

::: tip 목록과의 차이점
목록은 값을 나중에 바꿀 수 있지만, 튜플은 한 번 생성되면 값을 수정하거나 추가, 삭제할 수 없습니다. 데이터가 변하지 않음을 보장하고 싶을 때 튜플을 사용하세요. :::
