# 값 종류

프로그래밍을 하다 보면 어떤 변수에 들어있는 값이 숫자인지, 문자인지 확인해야 할 때가 있습니다. 달빛약속에서는 `값 종류` 문법을 사용하여 값의 타입을 확인할 수 있습니다.

## 기본 문법

값 뒤에 `의 값 종류`를 붙이면 해당 값의 종류를 문자열로 반환합니다.

<code-runner code='변수 의 값 종류' />

## 예시

각 데이터 타입에 따라 다음과 같은 결과를 반환합니다.

<code-runner :code='`10 의 값 종류 보여주기
"안녕" 의 값 종류 보여주기
참 의 값 종류 보여주기
[1, 2, 3] 의 값 종류 보여주기`' />

실행 결과는 각각 **"숫자"**, **"문자"**, **"참거짓"**, **"목록"**이 됩니다.

## 활용하기

값의 종류에 따라 서로 다른 처리를 하고 싶을 때 유용하게 사용할 수 있습니다.

<code-runner :code='`값 = 100
만약 값 의 값 종류 == "숫자" 이면
    "숫자입니다" 보여주기
아니면
    "숫자가 아닙니다" 보여주기`' />

## 직접 해보기

변수 `입력`에 들어있는 값이 "문자"인지 확인하여, 문자라면 "문자입니다"를 출력하고 아니면 "문자가 아닙니다"를 출력하는 코드를 작성해보세요.
