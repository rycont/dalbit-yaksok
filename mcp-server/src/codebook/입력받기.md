### 입력받기(Input)

```
이름 = 입력받기
나이 = "나이가 어떻게 되세요?" 입력받기
```

### 파이썬에서의 equivalent

```
이름 = input()
나이 = input("나이가 어떻게 되세요?")
```

### 주의

-   질문이 있는 경우: 질문 문자열 뒤에 공백 1칸 후 `입력받기`를 씁니다.
-   질문이 없는 경우: 바로 `입력받기`를 사용합니다.
-   입력받은 값은 항상 문자열로 반환됩니다.

### MCP 서버에서 사용하기

MCP 서버의 `execute` 도구를 사용할 때, `stdinInputs` 파라미터를 통해 입력값을 미리 제공할 수 있습니다:

```json
{
    "code": "이름 = '이름을 입력하세요: ' 입력받기\n이름 보여주기",
    "stdinInputs": ["홍길동"]
}
```

여러 번의 `입력받기` 호출이 있는 경우, `stdinInputs` 리스트의 값들이 순서대로 사용됩니다:

```json
{
    "code": "이름 = '이름: ' 입력받기\n나이 = '나이: ' 입력받기\n이름 + '님은 ' + 나이 + '세입니다' 보여주기",
    "stdinInputs": ["홍길동", "30"]
}
```
