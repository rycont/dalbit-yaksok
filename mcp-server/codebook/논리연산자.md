### 논리 연산자(Logical Operator)

```
참 이고 거짓
참 이거나 거짓

진수_밥_먹었다 = 참
진수_물_먹었다 = 참
진수_밥_먹었다 고 진수_물_먹었다
진수_물_먹었다 거나 진수_밥_먹었다

!진수_물_먹었다 보여주기
진수_밥_먹었다 아니다 보여주기
```

### 파이썬에서의 equivalent

```
True and False
True or False

진수_밥_먹었다 = True
진수_물_먹었다 = True
진수_밥_먹었다 and 진수_물_먹었다
진수_밥_먹었다 or 진수_물_먹었다

print(not 진수_물_먹었다)
print(not 진수_밥_먹었다)
```

### 연관 키워드

-   논리, 아니다, and, or, not, 이고, 고, 이거나, 거나, !
