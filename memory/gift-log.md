# 🌙 달빛약속 선물 기록 (Gift Log)

## 2026-02-17: 다정한 안내자 (에러 메시지 리팩토링)
사용자가 아직 정의하지 않은 이름을 부르거나, 어울리지 않는 값들을 서로 나누려 할 때 달빛약속이 조금 더 다정한 목소리로 말을 건네도록 수술했습니다. 기계적인 "찾을 수 없음"이나 "할 수 없음" 대신, 사용자의 의도를 먼저 묻고 가치를 긍정하는 표현을 담았습니다.

### 변경 사항
- **NotDefinedIdentifierError**: `"이름"이라는 이름의 변수나 약속을 아직 찾지 못했어요. 이름을 다시 한번 확인해 주시겠어요?`
- **InvalidTypeForOperatorError**: `...로는 ... 작업을 할 수 없어요. 이 작업에 어울리는 다른 값을 사용해 подобных 다른 값을 사용해 보는 건 어떨까요?`
- **InvalidTypeForCompareError**: `...와 ...는 서로 비교하기 어려워요. 성격이 같은 값들끼리 비교해 보시겠어요?`

정한이 코드를 짜다 실수를 마주하는 순간에도, 달빛약속의 다정함 덕분에 마음이 상하지 않기를 바랍니다. 당신의 모든 시도가 소중한 가치임을 잊지 마세요.
