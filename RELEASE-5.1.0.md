# 달빛약속 v5.1.0

## 이벤트 안에서 바깥 변수를 사용할 수 있어요

```
햄스터상태 = "정지 중"

@마이크로비트 양쪽 버튼 눌렀을 때
    만약 햄스터상태 == "정지 중" 이면
        햄스터상태 = "가는 중"
```

이전에는 이벤트 블록 안에서 바깥에 선언한 변수를 읽거나 바꾸면 `변수를 찾을 수 없어요` 에러가 났습니다. 이제 정상 동작합니다.

## 셀 모드에서 변수에 조사를 붙여 다른 모듈 명령에 쓸 수 있어요

```
셀 1: 과일 = ["사과", "바나나", "딸기"]
셀 2: @그래프 과일로 원그래프 그리기
```

이전 셀에서 만든 변수에 `로`, `을` 같은 조사를 붙여 `@모듈` 명령에 전달하면 인식하지 못하던 문제를 수정했습니다.

## 이벤트 구독 시 Scope 정보 전달

외부에서 `eventCreation.sub()`으로 이벤트를 구독할 때, 4번째 인자로 `Scope` 객체를 받을 수 있습니다. 이벤트가 등록된 코드 파일 등의 컨텍스트를 확인할 수 있습니다.
